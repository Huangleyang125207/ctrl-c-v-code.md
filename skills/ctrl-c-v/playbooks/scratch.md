# Playbook: Writing from scratch

> Loaded by § 5. This should be rare.

## Before you start

Confirm steps 0–5 truly returned nothing.
If uncertain, ask: "No existing pattern for [X]. Write from scratch?"

## Rules

- Minimum code. Today's problem only
- No speculative features or single-use abstractions
- 200 lines that could be 50 → rewrite to 50

## The tax

1. Add PATTERN / USE WHEN / COPY THIS tags to the code
2. Save a copy to `~/.claude/patterns/<type>-<variant>.md`
3. Update `~/.claude/patterns/INDEX.md`

This code is now a template. Next similar task = copy + change 5 lines.
